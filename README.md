## Deployment
This is a **Static Site**. It consists only of HTML, CSS, and JavaScript files.

### Deploying on Render
1. Create a new **Static Site** on Render.
2. Connect your GitHub repository.
3. Use the following settings:
   - **Build Command**: (Leave empty)
   - **Publish Directory**: `.` (Current directory)

### Deploying on Netlify / Vercel / GitHub Pages
- **Socials/Share**: Works perfectly on any static hosting platform.
- **Note**: Ensure `music.mp3` and `video.mp4` are uploaded with your code for the media to work.
